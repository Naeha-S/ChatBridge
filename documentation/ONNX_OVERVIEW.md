# ONNX Overview

This document provides an overview of ONNX integration in ChatBridge, including model loading, inference, and supported features.

- ONNX models are stored in the `onnx/` directory.
- Supported for fast inference and cross-platform compatibility.
- See `models/minilm/` for example ONNX model files.
- For developer usage, see API_REFERENCE.md and ARCHITECTURE.md.
