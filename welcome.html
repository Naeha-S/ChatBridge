Common questions

Do I need API keys to use ChatBridge?

Yes ‚Äî ChatBridge uses your own API keys to call AI models. Both Gemini and HuggingFace have generous free tiers. You can also add an OpenAI key for additional features.

Which AI platform should I start with?

Any of them. ChatBridge works on 25+ AI chat sites. Just navigate to one, click the ‚ö° button, and you're in. ChatGPT and Claude are good starting points.

Is my data sent to ChatBridge servers?

No. There are no ChatBridge servers. All data is stored locally in your browser. API calls go directly from your browser to the AI providers (Google, HuggingFace, OpenAI).

Can I export my conversation history?

Yes. You can export all saved conversations as JSON or Markdown at any time. The data is always yours to take with you.

Does it work on mobile?

ChatBridge is a Chrome desktop extension. It does not work on mobile browsers since they don't support extensions.
