FAQ

What people ask first

Do I need API keys?

Yes. ChatBridge uses your own Gemini and or Hugging Face credentials instead of routing through a hosted backend.

Is my data sent anywhere else?

No ChatBridge server sits in the path. Calls go directly from the extension to configured AI providers.

Can I export my chats?

Yes. Sessions can be exported whenever you want as JSON or Markdown.

Does it work on mobile?

No. The extension is built for desktop Chrome environments.
