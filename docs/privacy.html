Privacy Policy

Last updated: July 5, 2025  ·  Questions? getchatbridge@gmail.com

The short version

ChatBridge is a local-first Chrome extension. By default, your conversations never leave your browser. No accounts. No tracking. No ads.

What data we collect

By default, we collect nothing. All conversation data is stored exclusively in your browser's local storage (chrome.storage.local) and IndexedDB.

If you optionally enable the Cloud Gateway feature:

  • API requests are proxied through our Cloudflare Workers endpoint.
  • Requests are not logged or stored by us beyond the duration of the request.
  • You choose which AI provider API key to use — we never store your keys.

Data storage

All conversations, settings, and preferences are stored locally in your Chrome browser profile. This data is never synced to any ChatBridge server and is fully under your control. You can delete all data at any time from the Options → History page.

Third-party AI services

When you use ChatBridge to send messages to AI providers (OpenAI, Google, Anthropic, etc.), those messages go directly from your browser to those providers. Each provider's own privacy policy applies. ChatBridge does not intercept or store the content of those messages.

Telemetry

ChatBridge includes an opt-in only basic error telemetry system. It is disabled by default. If enabled, only anonymised error events are sent — never conversation content. You can disable this at any time from the Options → Privacy & Telemetry section.

Cookies

ChatBridge as a Chrome extension does not use browser cookies. The extension stores data using the Chrome Extensions storage APIs only. See our Cookie Preferences page for more information.

Children's privacy

ChatBridge is not directed at children under the age of 13. We do not knowingly collect personal information from children.

Changes to this policy

We may update this policy from time to time. Changes will be reflected by an updated date at the top of this page. For significant changes, we will notify users through the extension.

Contact

If you have any questions about this Privacy Policy, please contact us at getchatbridge@gmail.com.
