About cookies & storage

ChatBridge is a browser extension, not a website. It does not use HTTP cookies. Instead, it uses the Chrome Extensions storage APIs (chrome.storage.local) and browser IndexedDB to store your conversations and preferences entirely on your device.

This page describes the types of storage used and gives you control over each category.

Essential storage

Required for the extension to function. Stores your settings, API configuration, and conversations locally. Cannot be disabled.

Always on

Error telemetry

Anonymised crash and error reports to help us fix bugs. No conversation content is ever included. Disabled by default.

Off

Analytics

Aggregate, non-personal usage patterns (e.g. which sections of settings are opened) to improve the product. No personal data or AI content is tracked.

Off
